지금부터는 User Prompt는 그대로 두고, **System Prompt만 바꿔서** 모델의 답변이 어떻게 달라지는지 실험합니다.

- 맞춤법 검사기
- 감성 시인
- JSON 추출기

위 3가지를 버튼으로 바로 적용해 볼 수 있고, 직접 프롬프트를 수정해서 자유 실험도 가능합니다.

## 실습 목표

1. System Prompt가 모델의 말투와 형식을 얼마나 강하게 제어하는지 확인하기
2. 같은 입력이라도 역할 지시가 바뀌면 출력 스타일이 크게 달라진다는 점 체감하기
3. 형식 강제(JSON 등)를 통해 출력 안정성을 높이는 방법 익히기
